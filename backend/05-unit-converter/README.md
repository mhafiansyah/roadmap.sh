## Unit Converter
a sample solution for the [Unit Converter](https://roadmap.sh/projects/unit-converter) challenge from [roadmap.sh](roadmap.sh)

# How to run
1. Clone the repository
```
git clone https://github.com/mhafiansyah/roadmap.sh.git
```

2. navigate to the project directory
```
cd backend\05-unit-converter
```

3. install required dependencies
```
npm install
```

# Usage
1. run the index.ts in src
```
npx tsx src/index.ts
```

2. open index.html in root folder
